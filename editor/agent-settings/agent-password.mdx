### How to Set a Password

1. Go to the **Configuration** section.
2. Click on the **Password** button.

<Frame style={{ maxWidth: '400px' }}>
  <img src="/images/agent-settings/agent-password.png" alt="Release0.com – Agent password button" />
</Frame>

3. Enter the desired password in the dialog that appears.

<Frame style={{ maxWidth: '400px' }}>
  <img src="/images/agent-settings/agent-password-dialog.png" alt="Release0.com – Agent password dialog" />
</Frame>

### What Users Will See

Once the password is set, users visiting the agent will encounter a secure entry screen where they must enter the password to proceed.

<Frame style={{ maxWidth: '400px' }}>
  <img src="/images/agent-settings/agent-password-protect-page.jpg" alt="Release0.com – Password protected page" />
</Frame>

> You can update or remove the password at any time from the same **Configuration** section.
