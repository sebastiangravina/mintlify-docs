An Agent flow can be exported to a JSON file using the **menu button** at the top-right corner of the builder.


## Exporting an Agent Flow

1. Open the builder for the Agent you want to export.
2. Click the **menu button** (three dots) in the top-right corner.
3. Select **Export** from the dropdown menu.
4. Save the JSON file to your preferred location.

## Importing an Agent Flow

An exported Agent flow can be imported when creating a new Agent. Follow these steps:

1. Navigate to the Agent creation page.
2. Select the option **Import a file**.
3. Upload the previously exported JSON file.
4. Review the imported flow to ensure it matches your expectations.


### Validation and Testing

Before deploying the imported Agent, ensure:

- All flows are correctly mapped and functional.
- No missing or improperly linked nodes are present.
- The Agent dialog behaves as expected during testing.

By following these steps, you can effectively manage and reuse Agent flows in your projects.
